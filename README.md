## 기술

- Language: TypeScript
- Library: React
- UI-tools: MUI
- Asynchronous-State-Management: Tanstack Query V4
- Bundler: Vite
- Tools: Eslint, Prettier, Husky
- Package-Manager: Yarn 1

## 설정

아래 스크립트 실행하여 필요한 환경변수 설정

```bash
yarn setup:env
```

## 사용

서버 코드 실행

```bash
yarn server
```

클라이언트 코드 실행

```bash
yarn dev
```
